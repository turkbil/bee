Kurulum

  • Central'a baglanti ve aktivasyon
  • Otomatik migration sync

  • Git-based kod dagitimi

  • Versiyonlama ve rollback

Isletme
  • Merkezi dashboard (tum satellite'lar)
  • Kullanim bazli faturalandirma
  • SLA izleme ve raporlama

Ornek Konfigurasyon

Satellite node .env dosyasi ornegi:

# Satellite Node Configuration
TUUFI_MODE=satellite
TUUFI_CENTRAL_URL=https://tuufi.com
TUUFI_LICENSE_KEY=sat_gazette_xxxx
TUUFI_TENANT_ID=5001

# Local Resources
DB_CONNECTION=mysql
DB_HOST=localhost
DB_DATABASE=gazette_production

# Central Sync Settings
TUUFI_SYNC_INTERVAL=60
TUUFI_HEARTBEAT_INTERVAL=30
TUUFI_UPDATE_CHANNEL=stable

Oneriler

Hemen Baslayabilecekleriniz

  • Tenant tablosuna deployment_type enum ekle (shared, dedicated_db, satellite)
  • License API endpoint'lerini tanimla (/api/license/verify, /api/license/heartbeat)
  • Mevcut bootstrapper'lara satellite mode destegi ekle

Orta Vadede

  • Git-based guncelleme mekanizmasi (private repo + deploy key)
  • Merkezi monitoring dashboard (Grafana + tenant metrikleri)
  • Satellite installer script ve dokumantasyon
