Tarz Sayfasından Seçimler (Öncelikli)

Ana Tasarım Türü

Minimal tasarım yaklaşımı: Sade, temiz, gereksiz öğelerden arındırılmış. Beyaz alan bolca kullanılıyor, odak içerikte.

Alternatif Tarz

Retro / Vintage: Sepia tonları, film grain efekti, nostaljik his. Minimal ile harmanlayarak modern-vintage hibrit oluşturuldu.

Tipografi

Space Grotesk (Google Fonts) - Fütüristik, teknoloji ve AI projeleri için ideal. Minimal estetiğe uygun geometrik yapı.

Navigasyon

Bottom Navigation: Mobil-öncelikli, sabit alt bar. Tüm cihazlarda kolay erişim, modern UX.

Hero Tarzı

Split Hero: Sol %60 metin, sağ %40 görsel. Klasik kurumsal sunum, güçlü görsel hiyerarşi.

Kart Tasarımı

Özellik Kartı: Hizmetler için ikon + başlık + açıklama + CTA yapısı. Hover efektleri ile interaktif.

Özel Notlar

  • ✓ Dark/Light Mode: Sistem tercihine göre varsayılan, toggle butonu header'da. localStorage ile tercih kaydediliyor.
  • ✓ Vintage Texture: CSS SVG ile oluşturulan subtle noise efekti, vintage his katıyor.
  • ✓ AOS Animasyonları: Scroll-triggered fade-up/fade-right/fade-left animasyonları.
  • ✓ Retro Underline: Başlıklarda terracotta renkli accent underline.
  • ✓ Grayscale Effect: Hero görselinde grayscale, hover'da renkli geçiş.
  • ✓ Bottom Nav Active State: Scroll spy ile aktif section'ı vurgulama.
  • ⚡ Fallback Görseller: Unsplash placeholder kullanılıyor. Leonardo.ai ile üretilen görseller images/ klasörüne eklenmeli.

Kullanılan Teknolojiler

Tailwind CSS v4 (CDN) Space Grotesk Font FontAwesome 6.5 AOS (Animate On Scroll) Vanilla JavaScript CSS Custom Properties
