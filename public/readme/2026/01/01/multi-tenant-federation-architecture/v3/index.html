License Server'a Ulasilamazsa?

Grace Period (Tolerans Suresi)

  • Son basarili dogrulama 7 gun gecerli
  • License server 1 hafta down olsa bile site calisir
  • Lisans bilgisi local cache'te saklanir

Nasil Calisir?

// Her saat
$license = License::verify();
// Basarili → cache'e yaz
Cache::put('license', $license, 7 days);
// Basarisiz → cache'ten oku
$license = Cache::get('license');

Sik Sorulan Sorular

Gazete kendi sunucusunda ne gormek istiyor?

Kendi domain'inde calisan, kendi veritabanini kullanan, kendi dosyalarini barindiran tam bagimsiz bir site. Sadece lisans kontrolu icin bize baglanir.

Biz kod gelistirince ne olacak?

Git'e push ederiz → License server'a yeni release olarak koyariz → Musterilere bildirim gider → Onlar kendi zamanlarinda gunceller.

Modulleri nasil yonetecegiz?

Lisans bazli. Gazete "Blog + Shop" almis → Sadece o moduller aktif. License server hangi modullerin acik oldugunu bildirir.

Database migration nasil olacak?

Her guncellemeyle birlikte migration dosyalari gelir. Musteri php artisan migrate calistirir veya installer otomatik yapar.
