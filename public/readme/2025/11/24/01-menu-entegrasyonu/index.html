Ana Firma Oluşturma Akışı

  1. Admin "Yeni Ana Firma" butonuna tıklar
  2. Modal açılır
  3. Mevcut kullanıcıyı arar ve seçer
  4. Firma adı girer
  5. "Oluştur" → Kayıt oluşur
  6. Kurumsal kod otomatik: MZB-XXXXXXXX

Şube Bağlama Akışı

  1. Admin "Şube Bağla" butonuna tıklar
  2. Modal açılır
  3. Ana firmayı seçer (dropdown)
  4. Şube sahibi kullanıcıyı seçer
  5. Şube adı girer
  6. "Bağla" → parent_id ile kayıt oluşur
