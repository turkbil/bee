Basit Lisans Sistemi (3 Bileşen)

1

Lisans Anahtarı

Her müşteriye benzersiz bir anahtar verilir

TUUFI-GZTE-2026-XXXX-XXXX
  • • Müşteri .env dosyasına ekler
  • • UUID + checksum formatı
  • • Domain'e bağlı (opsiyonel)
2

Doğrulama API

Günde 1 kez lisans kontrolü yapılır

POST tuufi.com/api/license/verify
  • • Başarılı → 7 gün cache
  • • Başarısız → Uyarı göster
  • • 7 gün offline çalışabilir
3

Güncelleme Sistemi

Müşteri IT ekibi güncelleme yapar

git pull && php artisan migrate
  • • Private Git repo erişimi
  • • Veya ZIP indirme linki
  • • Versiyon kontrolü API'den

Müşteri Kurulum Adımları (Self-Hosted)

1

Lisans Satın Alma

Müşteri lisans satın alır, biz sisteme kaydederiz. Lisans anahtarı mail ile gönderilir.

2

Sunucu Hazırlığı

Müşteri IT ekibi sunucuyu hazırlar: PHP 8.2+, MySQL 8+, Redis, Nginx/Apache. Gereksinim listesi dokümanda verilir.

3

Kod Kurulumu

Private Git repo'ya erişim verilir veya ZIP indirilir. composer install çalıştırılır.

4

.env Ayarları

Lisans anahtarı ve database bilgileri girilir: 

TUUFI_LICENSE_KEY=GZTE-2026-XXXX-XXXX
TUUFI_LICENSE_SERVER=https://tuufi.com/api/license
DB_DATABASE=gazete_db
DB_USERNAME=gazete_user
5

Aktivasyon

php artisan tuufi:activate komutu çalıştırılır. Sistem lisansı doğrular ve aktif olur.

Hazır!

Sistem çalışır durumda. Günlük cron job lisansı otomatik doğrular. Güncelleme bildirimleri admin panelde görünür.

Güncelleme Stratejisi

Shared Müşteriler

  • Otomatik güncelleme
  • Tek kod tabanı, herkes güncellenir
  • Müşteri bir şey yapmaz
# Biz yaparız
git pull && php artisan migrate

Self-Hosted Müşteriler

  • Admin panelde "Güncelleme Mevcut" uyarısı
  • IT ekibi manuel günceller
  • Changelog linki verilir
# Müşteri IT ekibi yapar
git pull origin main
composer install --no-dev
php artisan migrate --force

Önerilen Fiyatlandırma Modeli

Müşteri Tipi Sunucu Lisans Modeli Destek Örnek
SHARED Bizim sunucu Aylık abonelik (düşük) Email ixtif.com
SELF-HOSTED Müşteri sunucusu Yıllık lisans (yüksek) Email + Öncelikli Gazete
ENTERPRISE Müşteri datacenter Yıllık + SLA 7/24 + Telefon Hastane
SCALING Başla shared → taşın Aylık → Yıllık geçiş Migration desteği Muzibu

İpucu: Self-hosted müşteri zaten sunucuya 500-1000$/ay veriyor. Senden sadece yazılım lisansı alıyor. Bu yüzden lisans ücreti shared'dan yüksek olabilir.
