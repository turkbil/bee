3 Anasayfa SEO Plani (1. Oncelik)

Ne Yapilacak?

Anasayfa icin Google'da gozukecek baslik, aciklama ve sosyal medya paylasim bilgilerini dolduracagiz. Birisi "Muzibu" diye arattiginda veya linki paylastiginda guzel gozukecek.

Doldurulacak SEO Alanlari

1. SEO Title (Baslik)

ZORUNLU

Google arama sonuclarinda gozuken baslik. 50-60 karakter ideal.

Ornek: "Muzibu - Turkiye'nin Muzik Platformu | Sarki Dinle"

2. SEO Description (Aciklama)

ZORUNLU

Google arama sonuclarinda basligin altinda gozuken aciklama. 150-160 karakter ideal.

Ornek: "Muzibu ile binlerce sarki dinle, playlist olustur, radyo yayinlari takip et. Turkiye'nin en yeni muzik platformu."

3. OG Title (Sosyal Medya Basligi)

ONERILEN

Facebook, Twitter, WhatsApp'ta paylasildinginda gozuken baslik.

Bos birakilirsa SEO Title kullanilir.

4. OG Description (Sosyal Medya Aciklamasi)

ONERILEN

Sosyal medyada paylasildinginda gozuken aciklama.

Bos birakilirsa SEO Description kullanilir.

5. OG Image (Paylasim Gorseli)

ONERILEN

Sosyal medyada paylasildinginda gozuken gorsel. 1200x630 piksel ideal.

Bos birakilirsa site logosu kullanilir.

6. Canonical URL

OTOMATIK

Sayfanin ana URL'i. Duplicate content sorunlarini onler.

https://muzibu.com/

7. Robots Meta

VARSAYILAN

Arama motorlarina sayfa hakkinda talimatlar.

index, follow (Varsayilan - degistirmeye gerek yok)
