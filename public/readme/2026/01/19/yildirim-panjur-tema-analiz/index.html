Kritik Eksiklikler

ONCELIKLI

1. Schema.org Markup Hatasi

Service sayfalarinda Schema type "Thing" olarak gosteriliyor. Google bunu dogru anlayamaz!

// YANLIS - Simdi
"@type": "Thing"

// DOGRU - Olmasi gereken
"@type": "Service" veya "LocalBusiness"

Dosya: Modules/Service/App/Http/Controllers/Front/ServiceController.php veya SEO component

2. LocalBusiness Schema Eksik

Yerel isletme bilgileri (adres, calisma saatleri, hizmet alani) Schema'da yok. Google Maps ve yerel arama icin kritik!

{
  "@type": "LocalBusiness",
  "name": "Yildirim Panjur",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "Istanbul",
    "addressRegion": "TR",
    "addressCountry": "TR"
  },
  "telephone": "+902125967230",
  "openingHours": "Mo-Su 00:00-23:59",
  "areaServed": {
    "@type": "City",
    "name": "Istanbul"
  },
  "priceRange": "$$"
}

3. FAQ Section ve FAQPage Schema

Service sayfalarinda FAQ bolumu yok. Google arama sonuclarinda "Sik Sorulan Sorular" snippet'i icin gerekli!

Her service icin 3-5 SSS eklenebilir. Schema ile zenginlestirilmeli.

Yuksek Oncelikli Iyilestirmeler

4. Iletisim Formu

Homepage ve iletisim sayfasinda form yok. Sadece telefon/WhatsApp var.

Oneri: "Ucretsiz Kesif Talep Formu" ekle - Ad, Telefon, Ilce, Mesaj alanlari

5. Musteri Yorumlari / Referanslar

Homepage'de musteri yorumu veya referans bolumu yok. Guven olusturmak icin kritik!

Oneri: ReviewSystem modulu veya statik testimonial slider ekle

6. Google Maps Entegrasyonu

Iletisim sayfasinda harita yok. "Istanbul Geneli" yaziyor ama gorsel yok.

Oneri: Hizmet verilen bolgeleri gosteren interaktif harita ekle

7. BreadcrumbList Schema

Breadcrumb gorunuyor ama Schema markup'i yok. Google arama sonuclarinda breadcrumb gostermez.

"@type": "BreadcrumbList" her sayfaya eklenmeli

Orta Oncelikli Iyilestirmeler

8. Back to Top Butonu

Uzun sayfalarda yukari don butonu yok

9. Cookie Consent

KVKK/GDPR uyumlulugu icin cookie banner gerekli

10. Lazy Loading

Gorsellerde loading="lazy" attribute eksik

11. Ozel Tema Dosyalari

Blog/Service icin t-3 ozellestirmesi yok, simple fallback kullaniyor

12. Social Share

Blog ve service sayfalari sosyal medya paylasim butonu yok

13. Print Stylesheet

Yazici dostu CSS yok (teklif/fatura ciktisi icin)

Performans Notlari

Tailwind CDN: Production'da CDN yerine build edilmis CSS kullanilmali

Console'da uyari gorunuyor: "cdn.tailwindcss.com should not be used in production"

CDN Bagimliliklari: AOS, GLightbox, FontAwesome harici CDN'lerden yukleniyor

Performans icin bundle edilebilir (ancak mevcut hali da kabul edilebilir)

PWA + instant.page: Dogru yapilandirilmis, calisiyor
