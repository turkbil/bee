Muzibu Responsive Sidebar Analizi

Hangi çözünürlükte hangi sidebar görünüyor? Detaylı breakpoint haritası.

Basit Anlatım (Herkes İçin)

Muzibu'da 3 ana bölüm var: Sol menü (navigasyon), Ana içerik (şarkılar/playlistler), Sağ panel (yeni/popüler şarkılar).

  • Telefonda (0-767px): Sadece ana içerik görünür. Sol menü hamburger butonu ile açılır.
  • Tablette (768-1023px): Aynı telefon gibi, hamburger menü kullanılır.
  • Küçük Laptop (1024-1279px): Sol menü görünür, sağ panel hala gizli.
  • Büyük Ekran (1280px+): Her üç bölüm de görünür - tam deneyim!

Teknik Detaylar (Geliştiriciler İçin)

Tailwind Breakpoint'ler:

  • sm: 640px
  • md: 768px
  • lg: 1024px (Sol sidebar açılır)
  • xl: 1280px (Sağ sidebar açılır)
  • 2xl: 1536px

Sağ Sidebar - Hangi Sayfalarda Görünür?

Sağ sidebar sadece müzik sayfalarında görünür. Dashboard, Corporate, Profile gibi sayfalarda gizlenir.

Görünür Olduğu Sayfalar:

  • muzibu.home - Ana Sayfa
  • muzibu.songs.show - Şarkı Detay
  • muzibu.albums.* - Albümler
  • muzibu.artists.* - Sanatçılar
  • muzibu.playlists.* - Playlistler
  • muzibu.genres.* - Türler
  • muzibu.sectors.* - Sektörler
  • muzibu.radios.* - Radyolar
  • muzibu.search - Arama
  • muzibu.favorites - Favoriler
  • muzibu.my-playlists - Playlistlerim

Gizli Olduğu Sayfalar:

  • dashboard - Kullanıcı Paneli
  • corporate/* - Kurumsal Sayfalar
  • profile - Profil
  • subscription/* - Üyelik
  • login/register - Giriş/Kayıt
