Shop System V4 İçin Öneriler

ÖNERİ 1: 8 Content Variation Sistemi (ZORUNLU)

Sorun: Mevcut sistemde her özellik tek bir şekilde anlatılıyor. AI matching, farklı kullanıcı sorguları, SEO varyasyonları için yetersiz.

Çözüm: Her özellik için 8 farklı anlatım üret:

  • technical: Teknik detay (ölçümler, sayılar)
  • benefit: Müşteriye faydası (sonuç odaklı)
  • slogan: 3-5 kelime akılda kalıcı
  • motto: 2-3 kelime güçlü ifade
  • short: 10-15 kelime özet
  • long: 50-80 kelime detaylı açıklama
  • comparison: Alternatiflerle karşılaştırma
  • keywords: Arama terimleri

Örnek (Li-Ion Battery):

{
  "li-ion-battery": {
    "technical": "24V 20Ah Li-Ion batarya, 2000+ şarj döngüsü, BMS korumalı",
    "benefit": "Tam gün çalış, şarj bekleme. Kurşun-aside göre %40 daha hafif",
    "slogan": "Bir Şarj, Tam Gün İş!",
    "motto": "Güç Hiç Bitmesin",
    "short": "24V Li-Ion, 2000+ döngü, hafif",
    "long": "24V 20Ah kapasiteli lityum iyon batarya sistemi...",
    "comparison": "Li-Ion vs Kurşun-Asit: %40 daha hafif, 3x daha uzun ömür",
    "keywords": "li-ion, lityum iyon, şarj süresi, batarya ömrü"
  }
}
ÖNERİ 2: AI-Focused Keywords (3 Kategori) (ZORUNLU)

Sorun: Mevcut keyword'ler insan odaklı. AI bu keyword'lerle ürünü bulamıyor.

Çözüm: 3 kategori keyword sistemi:

  • primary: Ana arama terimleri (transpalet, elektrikli transpalet)
  • synonyms: Eş anlamlılar (palet taşıyıcı, palet kaldırıcı)
  • usage_jargon: Kullanım jargonu (palet jack, depo transpaleti)

Örnek:

{
  "keywords": {
    "primary": ["transpalet", "elektrikli transpalet", "li-ion transpalet", "1.5 ton transpalet"],
    "synonyms": ["palet taşıyıcı", "elektrikli palet kaldırıcı", "akülü transpalet"],
    "usage_jargon": ["palet jack", "transpalet makinesi", "depo transpaleti", "lojistik transpaleti"]
  }
}
ÖNERİ 3: One-Line Description (ZORUNLU)

Sorun: Ürün kartlarında başlık altında kısa açıklama yok. Kullanıcı ürünü anlamıyor.

Çözüm: 120-150 karakter kısa tanım ekle.

Örnek:

"one_line_description": "1.5 ton kapasiteli, Li-Ion bataryalı, hafif ve kompakt transpalet. Dar koridorlarda yüksek manevra kabiliyeti."
ÖNERİ 4: Primary Specs (4 Sabit Alan) (ÖNEMLİ)

Sorun: Karşılaştırma sayfasında hangi 4 özellik gösterilecek? Her PDF farklı alanları vurguluyor.

Çözüm: Kategori bazlı 4 sabit alan belirle.

Transpalet için:

{
  "primary_specs": {
    "capacity": "1500 kg",
    "battery": "24V/20Ah Li-Ion",
    "weight": "120 kg",
    "fork_length": "1150 mm"
  }
}
ÖNERİ 5: Normalization Fields (ÖNEMLİ)

Sorun: Aynı özellik farklı PDF'lerde farklı isimlerle ("Stabilizing wheel" vs "Castor wheels" vs "Floating wheels")

Çözüm: Normalizasyon alanları ekle:

{
  "stabilizing_wheel": {
    "value": true,
    "original_text": "Industrial floating stabilizing wheels",
    "_normalized_field": "has_stabilizing_wheel",
    "_normalized_value": true,
    "_aliases": ["castor wheels", "floating wheels", "stabilizing wheels"],
    "_comparison_label": "Stabilize Tekerlek"
  }
}
ÖNERİ 6: PDF Content Quality Score (SEÇİMLİK)

Sorun: Hangi PDF'ler kaliteli, hangiler eksik? Takip sistemi yok.

Çözüm: Her PDF için içerik skoru hesapla:

{
  "content_quality_score": {
    "total": 85,
    "breakdown": {
      "technical_table": 20,
      "why_series": 15,
      "use_cases": 15,
      "cost_saving": 10,
      "platform_design": 10,
      "options": 10,
      "features": 5
    },
    "missing_fields": ["competitive_advantages"],
    "warnings": ["Maliyet tasarrufu verisi eksik"]
  }
}
ÖNERİ 7: Eksik Veri Handling (KRİTİK)

Sorun: EPL185 gibi PDF'lerde marketing içeriği eksik. Karşılaştırmada boş alanlar oluşuyor.

Çözüm: 3 katmanlı handling sistemi:

  • Katman 1 (Teknik): Her zaman dolu, güvenle karşılaştır
  • Katman 2 (Opsiyonlar): Çoğunda dolu, eksik alanları işaretle
  • Katman 3 (Marketing): Eksik olabilir, "Veri Yok" + "Üreticiye Sor" butonu
// EPL185'te eksik alan:
{
  "cost_saving": null,
  "_missing": true,
  "_missing_reason": "PDF'de bu bölüm yok",
  "_user_action": "contact_supplier"
}
ÖNERİ 8: FAQ Minimum 12-15 (Kategorize)

Sorun: Mevcut FAQ'ler yetersiz ve kategorize değil.

Çözüm: Minimum 12-15 FAQ, 4 kategoride:

  • Teknik (4-5 soru): Kapasite, batarya, performans
  • Kullanım (3-4 soru): Hangi uygulamalar, nasıl kullanılır
  • Bakım (2-3 soru): Bakım periyodu, servis
  • Satın Alma (2-3 soru): Fiyat, garanti, teslimat
