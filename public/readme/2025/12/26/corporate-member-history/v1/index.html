Kurumsal Uye Dinleme Gecmisi

26 Aralik 2025 - Muzibu Kurumsal Panel Gelistirmesi

Basit Anlatim (Herkes Icin)

Ne yapildi?
Kurumsal hesap sahipleri artik kendi uyelerinin dinleme gecmisini gorebiliyor. Hangi uye, ne zaman, hangi sarkiyi dinlemis - hepsini detayli sekilde takip edebilirsiniz.

Nasil kullanilir?
Kurumsal Panel'e gidin → Bir uyeyi genisletin → "Dinleme Gecmisi" butonuna tiklayin.

Guvenlik:
Sadece kendi kurumunuzun uyelerini gorebilirsiniz. Baskalarinin verileri erisemez.

Teknik Detaylar (Gelistiriciler Icin)

Eklenen Route'lar

GET /corporate/member/{id}/history
GET /api/corporate/member/{id}/history

{id} = account_id (kurumsal hesap ID), user_id degil - guvenlik icin

Controller Metodlari

memberHistory(Request $request, int $id)
apiMemberHistory(Request $request, int $id)
findCorporateMember(MuzibuCorporateAccount $parent, int $id)

Guvenlik Onlemleri

  • Auth middleware - Giris yapilmadan erisim yok
  • Owner kontrolu - Sadece ana sube (parent_id=null) erisebilir
  • Uye dogrulama - parent_id eslesme kontrolu
  • URL'de account_id kullanimi - user_id ifsa edilmiyor
