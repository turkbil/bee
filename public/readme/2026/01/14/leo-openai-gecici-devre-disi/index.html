Leo AI & OpenAI Geçici Devre Dışı Planı

Müzik yükleme sürecinde AI maliyetlerini geçici olarak düşürme analizi

Basit Anlatım (Herkes İçin)

Ne yapılıyor?
Şarkı yüklendiğinde sistem otomatik olarak iki AI servisi kullanıyor:

  • Leonardo AI: Şarkı için otomatik kapak görseli üretiyor (görsel oluşturma = kredi harcanıyor)
  • OpenAI: Görsel için yaratıcı prompt (açıklama metni) üretiyor (metin üretme = kredi harcanıyor)

Geçici değişiklik ne yapacak?

  • Leonardo görsel boyutu küçülecek → Daha az kredi harcanacak (%60-70 tasarruf)
  • OpenAI prompt üretimi kapanacak → Şarkı başlığı doğrudan kullanılacak (tam tasarruf)

Neden "geçici"?
Bu değişiklikler config ayarı ile yapılacak, kod silinmeyecek. İstediğinizde tek satır değişiklikle eski haline dönebilirsiniz.

Önerilen Değişiklikler

Yöntem 1: Config Tabanlı Toggle (ÖNERİLEN)

En temiz ve geri alınabilir yöntem. Tek bir config değişikliği ile on/off.

// config/ai.php (yeni eklenecek)
'muzibu' => [
    'leo_image_enabled' => env('MUZIBU_LEO_IMAGE_ENABLED', true),
    'leo_image_size' => env('MUZIBU_LEO_IMAGE_SIZE', '512x512'), // küçük boyut
    'openai_prompt_enabled' => env('MUZIBU_OPENAI_PROMPT_ENABLED', false), // geçici kapalı
],
// .env dosyasına eklenecek
MUZIBU_LEO_IMAGE_ENABLED=true
MUZIBU_LEO_IMAGE_SIZE=512x512
MUZIBU_OPENAI_PROMPT_ENABLED=false

Orijinal Haline Geri Alma

Geçici süre bittiğinde .env dosyasını değiştirmek yeterli:

// .env (orijinal değerler)
MUZIBU_LEO_IMAGE_ENABLED=true
MUZIBU_LEO_IMAGE_SIZE=1280x800
MUZIBU_OPENAI_PROMPT_ENABLED=true
php artisan config:clear

Özet: Yapılacaklar

1
config/ai.php'ye muzibu section ekle
leo_image_size, openai_prompt_enabled ayarları
2
MuzibuLeonardoAIService.php güncelle
OpenAI prompt'u config ile kontrol et
3
GenerateAICover.php güncelle
Boyut ve prompt'u config'den oku
4
.env dosyasına geçici değerler ekle
512x512 boyut, OpenAI prompt kapalı
