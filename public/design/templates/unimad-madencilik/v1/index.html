Tasarım Promptu

Bu web sitesi aşağıdaki prompt ile oluşturulmuştur:

Proje: Unimad Madencilik - Mühendislik ve Danışmanlık Web Sitesi

Sektör: Madencilik, Mimarlık, Jeoteknik Mühendisliği

Tasarım Değerleri:

  • Kurumsal Mesaj: "Tecrübe ve bilgi birikimini güncel teknoloji ile birleştirmek"
  • Renk Kodlaması: Bakır-turuncu (madencilik), mavi (mimarlık), yeşil (jeoteknik)
  • Dark mode varsayılan tema
  • Modern minimal profesyonel tasarım

Teknolojiler:

  • Tailwind CSS v4 (CDN)
  • Alpine.js (interaktivite)
  • AOS (scroll animasyonları)
  • Swiper (referans slider)
  • FontAwesome Pro (ikonlar)
  • Montserrat + Inter (fontlar)

Ana Bileşenler:

  • Video hero section (overlay)
  • Mega menu (dropdown'lar ortalı)
  • 3 hizmet kartı (her biri farklı renk)
  • İstatistik counter'ları
  • Değerler section
  • Referans slider (Swiper)
  • İletişim formu
  • Zengin footer
  • Dark/Light mode toggle

Görsel Strateji:

Profesyonel endüstri fotoğrafyası (madencilik sahaları, mühendis ekipleri), teknik illüstrasyonlar, sıcak ton aydınlatma, güven ve yetkinlik vurgusu.

Bu prompt, Claude Code AI tarafından yorumlanarak yukarıdaki web sitesi tasarımı oluşturulmuştur.
