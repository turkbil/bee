1 SSL/TLS Ayarları

1.1 SSL Modunu Değiştir

Sol menüden şu yolu takip et:

SSL/TLS → Overview

Sayfada 4 seçenek göreceksin. Full (strict) olanı seç:

Off
Flexible
Full
Full (strict)

1.2 Always HTTPS Aç

Sol menüden şu yolu takip et:

SSL/TLS → Edge Certificates

Aşağı kaydır, "Always Use HTTPS" bul:

Always Use HTTPS
OFF
ON

Toggle'a tıkla, yeşil yapsın

1.3 Minimum TLS Version

Aynı sayfada biraz daha aşağıda "Minimum TLS Version" var:

Minimum TLS Version: ← Bunu seç

1.4 HSTS Aktifleştir

Aynı sayfada "HTTP Strict Transport Security (HSTS)" bul:

  1. 1 Enable HSTS butonuna tıkla
  2. 2 Max Age: 6 months seç
  3. 3 Include subdomains: ON
  4. 4 Save butonuna tıkla

SSL/TLS Ayarları Tamam!

2 Cache Ayarları

2.1 Tiered Cache Aç (Çok Önemli!)

Sol menüden şu yolu takip et:

Caching → Tiered Cache

"Enable Smart Tiered Cache Topology" bul:

Smart Tiered Cache
ON yap
Bu özellik sunucu yükünü %90'a kadar azaltır. Kesinlikle aç!

2.2 Always Online Aç

Sol menüden şu yolu takip et:

Caching → Configuration
Always Online ON yap

Sunucu çökerse eski sayfaları gösterir

2.3 Crawler Hints Aç

Aynı sayfada (Configuration) aşağıda:

Crawler Hints ON yap

Google daha hızlı indexler (SEO için iyi)

Cache Ayarları Tamam!

3 Performans Optimizasyonları

3.1 Polish (Görsel Sıkıştırma) - ÇOK ÖNEMLİ!

Sol menüden şu yolu takip et:

Speed → Optimization → Image Optimization

"Polish" bölümünü bul:

Polish:
Off Lossless Lossy
← Bunu seç
WebP: ON yap
Görseller otomatik %30-50 küçülür, kalite aynı kalır!

3.2 Auto Minify

Sol menüden şu yolu takip et:

Speed → Optimization → Content Optimization

"Auto Minify" bölümünde üçünü de aç:

JavaScript

ON

CSS

ON

HTML

ON

3.3 Brotli Sıkıştırma

Aynı sayfada (Content Optimization):

Brotli ON yap

3.4 Early Hints

Sol menüden şu yolu takip et:

Speed → Optimization → Protocol Optimization
Early Hints ON yap

3.5 HTTP/3 (QUIC)

Sol menüden şu yolu takip et:

Network
HTTP/2 ON (zaten açık olmalı)
HTTP/3 (with QUIC) ON yap

Performans Ayarları Tamam!

4 Güvenlik (WAF)

4.1 WAF Managed Rules (Hazır Koruma)

Sol menüden şu yolu takip et:

Security → WAF → Managed rules

Şu iki ruleset'i aktifleştir:

1. Cloudflare Managed Ruleset

Deploy butonuna tıkla

2. Cloudflare OWASP Core Ruleset

Deploy butonuna tıkla

ÖNEMLİ: İlk hafta sadece izle (Log mode). 1 hafta sonra Block moduna geç.

4.2 Super Bot Fight Mode

Sol menüden şu yolu takip et:

Security → Bots

"Configure Super Bot Fight Mode" butonuna tıkla:

Definitely automated Block
Likely automated Managed Challenge
Verified bots (Google, Bing) Allow

Güvenlik Ayarları Tamam!

6 Rocket Loader (Opsiyonel - Test Et!)

DİKKAT: Bu özellik bazen JavaScript sorunlarına yol açar. Muzibu player Alpine.js kullandığı için sorun çıkarabilir. Önce test et, sorun olursa kapat!

Sol menüden şu yolu takip et:

Speed → Optimization → Content Optimization
Rocket Loader ON yap (test için)

Test Adımları:

  1. 1 https://www.muzibu.com aç
  2. 2 Bir şarkı çalmayı dene
  3. 3 F12 → Console sekmesine bak
