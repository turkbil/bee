4 Systemd Servisler (KRITIK!)

Bu servisler sunucu restart olunca otomatik baslamali. Yoksa queue'lar calismaz!

Horizon Service Dosyasi: /etc/systemd/system/horizon.service

[Unit]
Description=Laravel Horizon Queue Manager
After=redis.service network.target

[Service]
Type=simple
User=WEB_USER
Group=WEB_GROUP
WorkingDirectory=/path/to/project
ExecStart=/usr/bin/php artisan horizon
ExecStop=/usr/bin/php artisan horizon:terminate
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target

Komutlar:

sudo systemctl daemon-reload
sudo systemctl enable horizon
sudo systemctl start horizon
