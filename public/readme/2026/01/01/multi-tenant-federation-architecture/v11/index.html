5

Secilen Yontemler

YENI - v11

Basit Anlatim

Endustri standartlarindan en uygun olanlari sectik. ionCube gibi loader gerektiren sistemler yerine kendi sifreli dosya sistemimizi kullanacagiz. Bu sayede musteri sunucusuna ek kurulum gerekmeyecek.

Kullanacagimiz Yontemler

1. Sifreli Cekirdek Dosya (core.enc)

ionCube yerine kendi sifreleme sistemimiz. AES-256 ile sifreli dosya, anahtar sunucudan gelir. Loader gereksiz!

2. Domain Locking

Lisans sadece kayitli domain'de calisir. musteri.com icin alinan lisans baska-site.com'da calismaz.

3. Online Activation + Heartbeat

Ilk kurulumda aktivasyon, sonra periyodik heartbeat. Microsoft/Adobe modeli.

4. Server-Side Feature Flags

Premium ozellikler sunucudan acilir. Lokal dosya degistirerek ozellik acilamaz.

5. Challenge-Response

Her dogrulamada sunucu farkli soru sorar. Kayitli cevaplarla bypass imkansiz.

6. Grace Period (7 Gun)

Internet kesilirse 7 gun offline calisir. JetBrains modeli. Sonra durdurur.

7. Coklu Kontrol Noktasi

Middleware, Provider, Scheduler, Controller - birini bypass etsen digeri yakalar.

8. Kod Butunlugu (Checksum)

Kritik dosyalarin hash'i sunucuda sakli. Degistirilirse tespit edilir.

Opsiyonel (Gelecekte Eklenebilir)

Hardware Fingerprint

Sunucu MAC/CPU ID ile baglama. Sunucu degisince lisans gecersiz. Esnek olmadigi icin opsiyonel.

ionCube Encoding

Bytecode sifreleme. Ekstra guvenlik katmani olarak eklenebilir ama loader kurulumu gerektirir.

Koruma Senaryolari

X Dosya degistirildi -> Hash uyusmaz -> Calismaz

X Dosya silindi -> Config yok -> Boot olmaz

X Sahte dosya -> Hash uyusmaz -> Anahtar yok

X Loader bypass -> Kritik config yok -> Calismaz
