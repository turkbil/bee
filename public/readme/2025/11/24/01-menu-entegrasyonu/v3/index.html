Akış

  1. Admin "Yeni Kurumsal Hesap" butonuna tıklar
  2. Modal açılır
  3. Mevcut kullanıcılardan birini seçer (arama ile)
  4. Firma adı girer
  5. Şube ise üst firmayı seçer
  6. Max üye sayısı belirler
  7. "Oluştur" → muzibu_corporate_accounts kaydı oluşur
  8. Kurumsal kod otomatik generate edilir

Kaydedilecek Veriler

  • user_id → Seçilen kullanıcı
  • parent_id → Üst firma (null = ana firma)
  • company_name → Firma adı
  • corporate_code → Otomatik (MZB-XXXXXX)
  • max_members → Max üye sayısı
  • is_active → true (default)
