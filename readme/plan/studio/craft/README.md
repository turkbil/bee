## 🚀 Getting Started

### **Prerequisites**
```bash
• Node.js 18+
• Laravel 10+
• React 18+  
• TypeScript 4.9+
```

### **Development Setup**
```bash
# Frontend setup
cd studio-editor/
npm install
npm run dev

# Laravel API
php artisan serve
php artisan queue:work
```

### **Widget Development**
```bash
# Create new widget
npm run generate:widget HeroSection

# Test widget
npm run test:widget HeroSection

# Deploy widget
php artisan widget:deploy HeroSection
```
