Teknik Notlar

Dark Mode Default: Premium hukuk bürosu hissi için koyu tema varsayılan olarak aktif

FontAwesome Pro 7 Local: /assets/libs/fontawesome-pro@7.1.0/css/all.css kullanıldı (CDN değil)

Icon Hover: Tüm kartlarda fat (thin) → fas (solid) geçişi JavaScript ile uygulandı

AOS Animasyonlar: Scroll-triggered fade animasyonları, once: true ile tek seferlik

Mobile Menu: Slide-from-right off-canvas menu, overlay ile

Gradient Text: Animated gradient (6s infinite) hero başlığında, static gradient diğer yerlerde

Gradient Border: Animated border (4s) founder card'da, static border diğer yerlerde

Responsive: Mobile-first approach, 6 breakpoint (sm, md, lg, xl, 2xl)

Theme Toggle: localStorage ile kullanıcı tercihi saklanıyor

Smooth Scroll: Anchor link'lerde 80px header offset ile
