Sorun Ne?

Muzibu'nun ana rengi "coral" (turuncu-kirmizi, #ff7f50). Ancak butonlarda farkli renkler kullanilmis:

  • Login butonu: Cok soluk/seffaf (beyaz %5 seffaflik)
  • Register butonu: Coral gradient (dogru!)
  • Kullanici avatari: Pembe gradient (coral degil)
  • Sidebar butonlari: Pembe gradient (coral degil)

Ne Yapilacak?

Tum butonlar ayni renk ailesini kullanacak. Boylece site daha profesyonel ve tutarli gorunecek.

Tailwind Config Guncelleme

muzibu-coral-light header'da kullaniliyor ama tanimli degil. Eklenmeli: 

// tailwind/tenants/tenant-1001.config.js
colors: {
    'muzibu-coral': '#ff7f50',
    'muzibu-coral-light': '#ff9966',  // YENi EKLENECEK
    'muzibu-coral-dark': '#e65c3e',   // YENi EKLENECEK
}
