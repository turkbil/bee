Oneriler

1

Dead Code Temizligi

Kaldirilacak ozelliklerin kodlarini tamamen sil

Temizlenecek:
  • keyboard.js dosyasi (318 satir) - tamamen sil
  • Crossfade fonksiyonlari (~300 satir)
  • Shuffle/Repeat fonksiyonlari (~100 satir)
  • player.blade.php comment'li kodlar
Tahmini kazanc: ~700 satir, ~25KB
2

B2B Mode (Kurumsal)

Repeat kalkmis ama B2B mode kaliyor mu?

b2bMode = true iken queue bitince basa doner (infinite loop).
Bu kurumsal muzik icin gerekli olabilir - kararinizi belirtin.
3

Gapless Playback (Gelecek)

Sarkiler arasi bosluksuz gecis - Crossfade yerine

Crossfade problemli, gapless daha stabil. HLS.js ile yapilabilir.
Spotify/Apple Music tarzi - kullanici fark etmez.
4

Picture-in-Picture (PiP)

Mini player - tab degistirince kucuk pencere

Chrome/Safari destekliyor. Kullanici baska tab'a gecince mini player acilir.
